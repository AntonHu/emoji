# Emoji快速查找工具

支持中文、拼音或首字母模糊查找，点击emoji即可直接复制。

### 依赖安装
```bash
  yarn
```
### PC Web端

```bash
 yarn dev:h5
```

![PC预览图](https://github.com/user-attachments/assets/423340fc-3f2c-40b4-be7b-e2749c4f3a72)

### 小程序端

```bash
 yarn dev:weapp
```
